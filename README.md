# ⚔️ Desafio: Classificador de Nível de Herói

Primeiro desafio de lógica de programação do Felipão da [DIO](https://www.dio.me/).

## 📌 Objetivo
O objetivo deste projeto foi aplicar conceitos fundamentais de programação para criar um sistema que classifica o nível de um herói com base na sua quantidade de Experiência (XP).

## 🛠️ Tecnologias Utilizadas
* **JavaScript** (Lógica principal)
* **Node.js** (Ambiente de execução)
* **Git & GitHub** (Versionamento de código)

## 🧠 Conceitos Aplicados
Para a construção deste projeto, utilizei:
- **Variáveis**: Para armazenar o nome e o XP.
- **Operadores de Comparação**: Para definir as faixas de nível.
- **Estruturas de Decisão (if/else)**: Para determinar qual mensagem exibir.
- **Laços de Repetição (for)**: Para permitir a classificação de múltiplos heróis em uma única execução.

## 📊 Tabela de Níveis
| XP | Nível |
| :--- | :--- |
| Menor que 1.000 | Ferro |
| 1.001 - 2.000 | Bronze |
| 2.001 - 5.000 | Prata |
| 5.001 - 7.000 | Ouro |
| 7.001 - 8.000 | Platina |
| 8.001 - 9.000 | Ascendente |
| 9.001 - 10.000 | Imortal |
| Maior que 10.001 | Radiante |
